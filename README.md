# Curso de Python y Programación Orientada a Objetos

## Descripción

Este repositorio recoge los ejercicios y prácticas que estoy realizando durante mi proceso de aprendizaje de Python y Programación Orientada a Objetos (POO).

El objetivo de este curso es adquirir una base sólida en el lenguaje Python, comprender los fundamentos de la programación y aprender a desarrollar soluciones utilizando buenas prácticas de programación orientada a objetos.

## Contenidos trabajados

Durante las distintas sesiones se están realizando ejercicios prácticos sobre:

* Variables y tipos de datos.
* Entrada y salida de información.
* Operadores aritméticos y lógicos.
* Estructuras condicionales (`if`, `elif`, `else`).
* Estructura `match`.
* Bucles `while`.
* Bucles `for`.
* Funciones.
* Listas y colecciones de datos.
* Programación Orientada a Objetos (POO).
* Clases y objetos.
* Herencia y reutilización de código.

## Objetivo del repositorio

Este repositorio sirve como seguimiento de mi evolución en Python, almacenando los ejercicios realizados durante el curso y las soluciones desarrolladas para reforzar los conceptos aprendidos en cada sesión.
